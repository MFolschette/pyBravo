BRAvo Biological netwoRk Assembly

Welcome to BRAvo, a web application that aims to assemble regulatory and signaling networks.

Objectives

With BRAvo, we propose a framework based on Semantic Web technologies. It addresses (i) the uniform identification of multi-source biological entities, (ii) the description of labeled directed graphs through RDF, and (iii) the use of BioPAX as a semantic reference.

Regulatory Network

A gene regulatory network is a set of genes that interact with each other to control a specific cellular function. BRAvo Regulatory Network is an interactive tool to assemble step by step upstream regulatory networks.

Signaling Network

A signaling network involves various biological entities (genes, proteins, small molecules) that are connected by reactions (transcription factors, catalysis, phosphorylation). BRAvo Signaling Network is an interactive tool to assemble step by step upstream signaling networks.

Batch Reconstruction

BRAvo batch reconstruction provides access to a command line tool to assemble full upstream regulatory networks.
