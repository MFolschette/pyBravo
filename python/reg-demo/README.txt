This folder contains data, scripts and results related to the notebook “BRAvo-regulation-demo.ipynb” in the parent folder.

Original folders:
  - data: input data for the notebook
  - scripts: Bash and Python scripts used by the notebook

Result folder created by the folder:
  - cytoscape: network (.sif) and table (.csv) files for Cytoscape
  - iggy: results from the Iggy execution (computation predictions)
